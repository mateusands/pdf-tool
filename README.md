# Gerenciador de PDF e Word (Tkinter)

Aplicação em Python com interface gráfica para:

- Dividir PDF
- Juntar PDFs
- Converter PDF para Word
- Converter Word para PDF

## Instalação

pip install -r requirements.txt

## Uso

python main.py

# Funcionalidades

Dividir PDF

Selecione um arquivo e informe páginas separadas por vírgula ou intervalo.

Juntar PDFs

Adicione vários PDFs e organize a ordem antes de juntar.

Converter


Converte PDF ↔ Word.
